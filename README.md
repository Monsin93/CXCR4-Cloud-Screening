# CXCR4-Cloud-Screening
This repository contains the predicted pharmacokinetic and physicochemical properties for a series of candidate compounds evaluated as potential CXCR4 antagonists. The data provided here supports the computational triage and lead-likeness assessment described in our study.
Repository Contentdata/: Contains Supplementary.csv which provides Molecular structure information and comprehensive ADME data.

Tools 
UsedSwissADME: For calculating physicochemical descriptors and predicting ADME parameters, druglikeness, and medicinal chemistry friendliness.
Rush Cloud Platform: The infrastructure used for the overall virtual screening workflow.
